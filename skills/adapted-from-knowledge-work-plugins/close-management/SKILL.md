# Close Management (OpenClaw Adapted)

Manage financial close processes efficiently.

## Month-End Close Checklist

```markdown
# Month-End Close: [Month YYYY]

**Close Date Target:** [Date]
**Actual Close Date:** [Date]
**Days to Close:** [X]

---

## Day 1-3: Pre-Close

- [ ] Review unreconciled transactions
- [ ] Request missing invoices/receipts
- [ ] Review accruals needed
- [ ] Notify vendors of payment schedules

## Day 4-7: Reconciliations

- [ ] Bank reconciliations complete
- [ ] Credit card reconciliations complete
- [ ] Intercompany reconciliations complete
- [ ] Accounts receivable aging review
- [ ] Accounts payable aging review

## Day 8-10: Adjustments

- [ ] Accrual entries recorded
- [ ] Prepaid/deferred entries recorded
- [ ] Depreciation calculated and recorded
- [ ] Inventory adjustments (if applicable)
- [ ] Foreign currency revaluation (if applicable)

## Day 11-12: Review

- [ ] Trial balance review
- [ ] Variance analysis complete
- [ ] Unusual items investigated
- [ ] Management review completed

## Day 13-15: Finalization

- [ ] Financial statements prepared
- [ ] Close period in accounting system
- [ ] Backup data/reports
- [ ] Distribute financial package
```
