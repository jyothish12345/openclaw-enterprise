# Audit Support (OpenClaw Adapted)

Prepare for and support financial audits.

## Audit Prep Checklist

```markdown
# Audit Preparation: [Year YYYY]

---

## Pre-Audit (4-6 weeks before)

- [ ] Review prior year audit findings
- [ ] Update accounting policies documentation
- [ ] Prepare PBC (Provided by Client) list
- [ ] Organize supporting documentation
- [ ] Reconcile all accounts
- [ ] Review significant transactions
- [ ] Document internal controls

## Documentation Needed

### Financial Records
- [ ] Trial balance
- [ ] General ledger detail
- [ ] Bank statements (all accounts)
- [ ] Reconciliations (all accounts)
- [ ] Financial statements (draft)

### Supporting Documents
- [ ] Material contracts
- [ ] Loan agreements
- [ ] Lease agreements
- [ ] Investment documentation
- [ ] Tax returns (prior year)

### Transaction Support
- [ ] Revenue support (invoices, contracts)
- [ ] Expense support (receipts, approvals)
- [ ] Capital expenditure documentation
- [ ] Payroll records

## Audit Response Template

# Audit Request Response

**Request #:** [##]

**Assigned to:** [Name]

---

[Auditor's request verbatim]

## Response

- [Document 2]

- [ ] Response prepared
- [ ] Reviewed by [Name]

```
