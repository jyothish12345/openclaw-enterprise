## SEO Fundamentals

### Keyword Strategy
1. **Primary keyword** â€” Use in: headline, first paragraph, one subheading, meta description, URL
2. **Secondary keywords (2-3)** â€” Use naturally in body and subheadings
3. **Write for humans first** â€” No keyword stuffing

### On-Page SEO Checklist
- [ ] Title tag: <60 chars, includes primary keyword
- [ ] Meta description: <160 chars, includes keyword
- [ ] URL slug: short, descriptive, keyword
- [ ] H1: one per page, matches title
- [ ] H2/H3: descriptive, include keywords naturally
- [ ] Image alt text: descriptive, keyword where relevant
- [ ] Internal links: 2-3 to related content
- [ ] External links: 1-2 to authoritative sources

## Headline Formulas

**How-To:**
- "How to [achieve result] [without obstacle]"
- Example: "How to Double Email Opens Without Sending More"

**Numbered List:**
- "[Number] [adjective] ways to [result]"
- Example: "7 Proven Ways to Reduce Churn"

**Contrarian:**
- "Why [common belief] is wrong (and what to do instead)"
- Example: "Why More Content Isn't the Answer"

**Guide:**
- "The [adjective] guide to [topic]"
- Example: "The Complete Guide to B2B Content Marketing"

**Do This, Not That:**
- "[Do this], not [that]"
- Example: "Build a Community, Not Just an Audience"

**What We Learned:**
- "What [impressive result] taught us about [topic]"
- Example: "What 10,000 A/B Tests Taught Us About Subject Lines"

## Call-to-Action Best Practices

### CTA Principles
- Use action verbs: Get, Start, Download, Join, Try, See
- Be specific: "Start your free trial" > "Submit"
- Create urgency (when genuine): "Join 500 teams already using this"
- Reduce risk: "No credit card required", "Cancel anytime"
- One primary CTA per page/email

### CTA Examples by Context

| Context | CTA Examples |
|---------|--------------|
| Blog | "Read our complete guide" / "Subscribe for weekly insights" |
| Landing page | "Start free trial" / "Get a demo" / "See pricing" |
| Email | "Read the full story" / "Claim your spot" / "Reply and tell us" |
| Social | "Drop a comment if you agree" / "Save for later" / "Link in bio" |
| Case study | "See how [product] works for you" / "Talk to our team" |

## Channel-Specific Tips

### Blog
- Subheading every 200-300 words
- Bullet points and lists
- One data point/example per section
- Front-load key information

### Social Media
- **LinkedIn:** Professional but human, 1,300 char sweet spot, personal stories
- **Twitter/X:** Concise, punchy, threads for longer content
- **Instagram:** Visual-first, line breaks, storytelling hooks
- **Facebook:** Conversational, under 80 chars for links

### Email
- Subject lines: urgency, curiosity, or clear value
- Personalize where possible
- Scannable: bold phrases, short paragraphs, bullets
- Mobile-first

### Landing Pages
- Lead with benefits, not features
- Use "you" language
- Answer "so what?" for every section
- Reduce friction near CTAs
