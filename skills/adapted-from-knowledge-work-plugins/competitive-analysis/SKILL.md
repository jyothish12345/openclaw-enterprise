# Competitive Analysis (OpenClaw Adapted)

Research and analyze competitors to inform strategy.

## When to Use
- Researching competitors
- Analyzing market positioning
- Identifying differentiation
- Tracking competitive moves
- Planning product/marketing strategy

## Competitive Analysis Template

```markdown
# Competitive Analysis: [Competitor Name]

**Date:** [YYYY-MM-DD]
**Analyst:** [Name]

---

## Company Overview

**Founded:** [Year]
**Size:** [Employees / Revenue estimate]
**Funding:** [Stage / Amount]
**Target market:** [Who they serve]

---

## Product/Service

**Core offering:**  
[What they sell]

**Key features:**
- [Feature 1]
- [Feature 2]
- [Feature 3]

**Pricing:**
| Tier | Price | Features |
|------|-------|----------|
| [Tier 1] | $[X]/mo | [Features] |

---

## Positioning

**Tagline/Value prop:** "[Their claim]"

**Messaging themes:**
1. [Theme 1]
2. [Theme 2]

**Target audience:** [Who they target]

---

## Marketing Strategy

**Channels:**
- [Channel 1]: [Activity level]
- [Channel 2]: [Activity level]

**Content approach:** [Strategy observed]

**Ad spend estimate:** [High/Medium/Low]

---

## Strengths

- [Strength 1]
- [Strength 2]

## Weaknesses

- [Weakness 1]
- [Weakness 2]

---

## Our Differentiation

**Where we win:**
- [Advantage 1]
- [Advantage 2]

**Where they win:**
- [Their advantage]

**Recommendation:** [Strategic insight]
```

## Research Using OpenClaw

```bash
# Web research
web_search "Company X product features"
web_search "Company X pricing"
web_fetch https://companyX.com

# Save findings
write to /memory/competitive-intel/company-x.md
```

## File Storage
`/memory/competitive-intel/[competitor-name].md`
